void graphene::chain::generic_witness_scheduler< WitnessID, RNG, CountType, OffsetType, debug >::update ( const T &  revised_set)
inline

Convenience function to call insert_all() and remove_all() as needed to update to the given revised_set.

This function calls find() on revised_set for all current witnesses. Running time is O(n*log(n)) if the revised_set implementation of find() is O(log(n)).

TODO: Rewriting to use std::set_difference may marginally increase efficiency, but a benchmark is needed to justify this.
