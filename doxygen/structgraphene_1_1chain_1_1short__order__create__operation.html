Detailed Description

Define a new short order, if it is filled it will be merged with existing call orders for the same account. If maintenance_collateral_ratio is set it will update any existing open call orders to use the new maintenance level.

When shorting you specify the total amount to sell and the amount of collateral along with the initial ratio. The price it will sell at is (amount_to_sell/(collateral*initial_collateral_ratio/2000))
